# KeyStorm — Persian Secure Password Generator

**رمزساز و مدیر رمز عبور فوق امن فارسی با تم سایبرپانک نئون**

A beautiful, offline, ultra-secure password generator & manager with full Persian RTL support and cyberpunk neon design.

## Features • امکانات
|-------|-------|
| Truly random & cryptographically secure passwords | تولید رمز کاملاً تصادفی و رمزنگاری‌شده |
| Only safe characters (accepted everywhere – banks, Instagram, Gmail…) | فقط کاراکترهای امن (همه سایت قبول می‌کنه) |
| One-click copy (modern clipboard API) | کپی با یک کلیک |
| Auto-save to local SQLite database | ذخیره خودکار در دیتابیس SQLite |
| Stunning neon cyberpunk UI with animations | تم نئون سایبرپانک + انیمیشن‌های خفن |
| Full Persian RTL + Vazirmatn font | کاملاً فارسی و راست‌چین |
| Fully responsive (mobile & desktop) | ریسپانسیو کامل |
